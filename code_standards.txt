Function Documentation as follows:

FILENAME
	FUNCTIONNAME( Parameters )
		Params
			parameter (type) (optional/required)
		Returns
			what the function returns
		Calls
			functionName (params)
			or
			table name - value
		Usage
			what the function does
		Scope
			local or global
